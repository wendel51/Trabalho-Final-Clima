Climatologia Aplicada ao Paraná:
Curitiba e o caso da seca de 2019 - 2020

O Que é este Projeto?

Este projeto busca relacionar dados climáticos para uma aplicação de caso concreta: A compreensão do fenômeno de estiagem prolongada e seca no Paraná durante o intervalo de 2019 - 2020.

Pretende-se, também e a partir desta análise, fomentar a discussão referente aos efeitos das mudanças climáticas para com a Capital paranaense.
